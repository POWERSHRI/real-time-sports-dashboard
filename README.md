# 🔥 Real-Time Sports Analytics Dashboard

## 📌 Overview
A full-stack web application that simulates real-time sports match updates using WebSockets.

This project demonstrates real-time data streaming, live analytics, and interactive dashboards without using external APIs.

---

## 🚀 Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Chart.js

### Backend
- Java 17
- Spring Boot 3
- WebSockets (STOMP + SockJS)
- Maven

---

## ⚡ Features

- 📡 Real-time match updates using WebSockets
- 📊 Live performance graphs
- 🧠 Simulated realistic cricket data
- 🎥 Dynamic video background UI
- 📈 Analytics (Run Rate, Score Progression)
- ⚡ No API required (custom data simulation)

---

## 🧠 How It Works

- Backend generates match data programmatically
- Updates every few seconds
- Sends data via WebSockets
- Frontend subscribes and updates UI instantly

---

## ▶️ Run Locally

### Backend
cd backend
mvn spring-boot:run

### Frontend
cd frontend
npm install
npm start
